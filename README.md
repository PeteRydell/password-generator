# password-generator

This password generator is intended to provide a multi-character, randomized password based on user input and declarations.

<img width="444" alt="Screen Shot 2019-10-05 at 9 21 51 PM" src="https://user-images.githubusercontent.com/31483933/66264537-f9f51580-e7bb-11e9-8ab7-37034b17c259.png">

The user is prompted to and provides an input determining the length of the password to be generated:
<img width="446" alt="Screen Shot 2019-10-05 at 9 21 10 PM" src="https://user-images.githubusercontent.com/31483933/66264530-e649af00-e7bb-11e9-8a3d-3a1ded7cc9e3.png">

The user is then presented a confirmation window to determine if he or she would like to include special characters:
<img width="442" alt="Screen Shot 2019-10-05 at 9 21 27 PM" src="https://user-images.githubusercontent.com/31483933/66264532-ee095380-e7bb-11e9-834d-786014d770f8.png">

The user is then presented with a confirmation window to determine whether he or she would like to include numeric characters:
<img width="1236" alt="Screen Shot 2019-10-05 at 9 20 47 PM" src="https://user-images.githubusercontent.com/31483933/66264535-f9f51580-e7bb-11e9-8105-cfb75115a030.png">

The user is then presented with a confirmation window to determine whether he or she would like to include lower case letters:
<img width="444" alt="Screen Shot 2019-10-05 at 9 21 39 PM" src="https://user-images.githubusercontent.com/31483933/66264536-f9f51580-e7bb-11e9-8a1d-4e4de3b08ebe.png">

The user is finally presented with a confirmation window to determine whether he or she would like to include upper case letters:
<img width="445" alt="Screen Shot 2019-10-05 at 9 22 06 PM" src="https://user-images.githubusercontent.com/31483933/66264538-f9f51580-e7bb-11e9-8293-8da1c3041b12.png">

These determinations depending on true or false are then run through a loop to provide a rnadom collection of characters and displayed in the text area on the web page:
![Picture1](https://user-images.githubusercontent.com/31483933/66264534-f95c7f00-e7bb-11e9-850c-ccd0cc8f6b19.png)
